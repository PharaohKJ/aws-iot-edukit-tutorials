Core2forAWS

The Core2 for AWS main library provides convenient wrapper functions to enable various hardware features. These wrapper functions perform the required initialization and configure the hardware with preset parameters. To enable this feature using KConfig, use the command pio run --environment core2foraws --target menuconfig from within the root of the project directory in your PlatformIO terminal window and go to the menu Component Config --> Core2 for AWS hardware enable to set the features you want to enable.

Example

The following example initializes the Core2 for AWS, which initializes the PMU for the battery and other enabled features â€” such as the display (and SPI bus). It then sets the screen brightness to 80.

#include "core2forAWS.h"
void app_main(void){
Core2ForAWS_Init();
Core2ForAWS_Display_SetBrightness(80); /* Should place this last to allow the display to finish initializing. */
}
